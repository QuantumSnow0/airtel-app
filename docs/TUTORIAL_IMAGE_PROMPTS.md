# Tutorial Image Prompts for Grok

Use these prompts to generate images for each tutorial slide. Each image should be:

- **Size**: 512x512px or 1024x1024px (square format)
- **Style**: Modern, clean, minimalist, app illustration style
- **Background**: Dark (#0A0A0A) or transparent
- **Format**: PNG with transparency preferred

---

## Slide 1: Welcome to Airtel Router

**Color**: #FFD700 (Gold/Yellow)

**Prompt:**

```
Create a modern, minimalist illustration of a golden rocket ship launching upward against a dark space background (#0A0A0A). The rocket should be sleek and futuristic, colored in vibrant gold (#FFD700) with subtle gradients. Include golden sparkles and stars trailing behind it. The rocket should be centered, taking up about 60% of the frame. Style: clean vector illustration, app icon aesthetic, professional and welcoming. No text, just the illustration.
```

**Alternative (simpler):**

A golden rocket icon (#FFD700) on dark background (#0A0A0A), modern flat design, minimalist, app illustration style, centered, clean lines, professional

## Usage Instructions

1. **Generate Images**: Use each prompt with Grok to generate the images
2. **Save Images**: Save as PNG files in `assets/images/tutorial/`:
   - `tutorial-welcome.png`
   - `tutorial-authentication.png`
   - `tutorial-ai-automation.png`
   - `tutorial-messages.png`
   - `tutorial-pin.png`
   - `tutorial-quick-reply.png`
   - `tutorial-chat-experience.png`
   - `tutorial-batch-operations.png`
   - `tutorial-notifications.png`
   - `tutorial-complete.png`

3. **Update Tutorial**: Replace icon components with Image components using these files
