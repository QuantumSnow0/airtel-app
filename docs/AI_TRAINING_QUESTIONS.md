# AI Training Questions - Airtel 5G Smart Connect

## Questions to Ask Agent

### 1. Installation Timeline & Process

- [ ] How long does installation take? (e.g., 1-2 hours?)
- [ ] How soon can installation be scheduled after ordering?
- [ ] What are the installation requirements/preparation needed?
- [ ] What equipment/tools does the installer bring?

### 2. Data Cap & Fair Usage

- [ ] What happens if customer exceeds 1TB monthly limit?
- [ ] Can customers purchase additional data top-ups?
- [ ] Does speed get throttled after 1TB, or does service stop?
- [ ] Is there a fair usage policy?
- [ ] Does unused data roll over to next month?

### 3. Equipment & Requirements

- [ ] What equipment is included in the package?
- [ ] Do customers need to buy anything else (cables, mounts, etc.)?
- [ ] What are the home/technical requirements for installation?
- [ ] Do customers need existing Airtel service or can anyone subscribe?

### 4. Package Changes

- [ ] Can customers upgrade or downgrade packages?
- [ ] How does upgrade/downgrade process work?
- [ ] Is there a fee for changing packages?
- [ ] When do package changes take effect?

### 5. Warranty & Support

- [ ] What's the warranty period for the router?
- [ ] What if the router breaks or has issues?
- [ ] Is there a replacement policy?
- [ ] What's covered under warranty vs what's not?
- [ ] How long does replacement take?

### 6. Contract & Cancellation

- [ ] Is there a minimum contract period?
- [ ] Can customers cancel anytime?
- [ ] What's the cancellation policy?
- [ ] Are there cancellation fees?
- [ ] What happens to the router if customer cancels?

### 7. Speed & Performance

- [ ] What speeds can customers actually expect? (Real-world vs advertised)
- [ ] Are speeds guaranteed or "up to"?
- [ ] What affects speed performance?
- [ ] Is there a speed test tool or way to verify speeds?

### 8. Multiple Locations & Transfers

- [ ] What if customer moves to a new location?
- [ ] Can router be transferred to new address?
- [ ] Is there a transfer fee?
- [ ] What if new location doesn't have coverage?

### 9. Business/Commercial Use

- [ ] Can router be used for business/commercial purposes?
- [ ] Are there different packages for business use?
- [ ] Any restrictions on commercial use?

### 10. Technical Support

- [ ] What technical issues can AI help with vs need agent?
- [ ] Common troubleshooting steps customers can try?
- [ ] How to reset router if needed?
- [ ] How to access router settings/admin panel?

### 11. Delivery & Setup

- [ ] How long does delivery take before installation?
- [ ] Is router delivered separately or brought by installer?
- [ ] What's included in the installation service?
- [ ] Do customers get setup instructions/manual?

### 12. Coverage & Service Quality

- [ ] What if coverage isn't good at customer's location after installation?
- [ ] Can customer get refund if service doesn't work?
- [ ] What's the service level agreement (SLA)?
- [ ] How to check coverage before ordering?

### 13. Payment & Billing

- [ ] What payment methods are accepted?
- [ ] When is monthly payment due?
- [ ] What happens if payment is late?
- [ ] Are there any hidden fees or charges?
- [ ] Do customers get invoices/receipts?

### 14. Additional Features

- [ ] What are the parental controls features?
- [ ] How many devices can connect simultaneously? (We know 64, but confirm)
- [ ] Can customers set up guest networks?
- [ ] Any mobile app for managing the router?

### 15. Promotions & Offers

- [ ] Are there any current promotions or discounts?
- [ ] Referral programs?
- [ ] Loyalty rewards for long-term customers?
